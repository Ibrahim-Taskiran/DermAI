# DermAI – AI-Based Skin Disease Detection Mobile Application

DermAI is a mobile-based skin disease analysis system developed as a software engineering term project.
The application allows users to upload or capture a skin image and receive AI-assisted predictions for possible skin conditions.

The system combines:

* Mobile application development
* Artificial intelligence and machine learning
* Image preprocessing
* Backend API communication

---

# Project Objective

The main objective of DermAI is to assist users by analyzing skin images and predicting possible dermatological conditions using machine learning models.

The application is designed as a decision-support system and does **not replace professional medical diagnosis**.

---

# Core Features

* Capture image using camera
* Select image from gallery
* Send image to backend server
* AI-based disease prediction
* Display Top-3 predicted diseases
* Show confidence scores
* Present care recommendations
* Medical warning suggestions

# System Workflow

1. User selects or captures a skin image
2. Mobile app sends image to backend API
3. Backend processes the image
4. AI model predicts disease probabilities
5. Results are returned to mobile app
6. User sees prediction and recommendations
